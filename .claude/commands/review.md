description: End-of-session review to capture learnings comprehensively
---

Perform a comprehensive end-of-session review:

1. **Accomplishments**: Summarize key work completed
2. **Learnings**: Extract what worked well and what didn't
3. **Patterns**: Document new patterns discovered
4. **Skills**: Identify skills to create or update
5. **Failures**: Document any failures for future reference
6. **Actions**: Create/update skills as needed
7. **Next Steps**: Suggest follow-up for future sessions

Use the @agentic-documenter agent for comprehensive analysis if the session was complex.
